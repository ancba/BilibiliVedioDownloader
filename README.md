# BilibiliVedioDownloader
Download Bilibili vedio bt wget

# 环境要求
* Python版本 >= 3
* 有良好的网络链接，且能够正常访问bilibili
* 系统中带有wget，且能够正常使用

# 安装与使用
```
git clone https://github.com/ancba/BilibiliVedioDownloader.git
cd BilibiliVedioDownloader
python main.py
```
清晰度
| 值 |对应的清晰度|
| --- | -----------|
| 16 | 360p        |
| 32 | 480p        |
| 64 | 720p 需登录 |
| 74 | 720p60 需要大会员 |
| 80 | 1080p需登录 |
| 112 | 1080p+ 需要大会员 |
| 116 | 1080p60 需要大会员 |
| 120 | 4k 需要大会员 |
